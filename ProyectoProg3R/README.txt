Para ejecutar la aplicación esta se ejecuta desde la clase VLogin.


La base de datos buena es la que se llama Secuoia asecas sin el .db.


Para comprar los productos tienes que seleccionar los que quieras con control + click y luego darle a confirmar.


Cada usuario tiene su propia carpeta con sus tickets.

Despues de hacer una compra, el programa te crea una carpeta con tu nombre de usuario dentro de la carpeta users. 
Para visualizarla hay que hacer refresh al proyecto desde eclipse


Para checkear tus tickets después de hacer una compra hay que pulsar el botón de history.
